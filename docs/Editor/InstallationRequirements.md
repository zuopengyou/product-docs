# 编辑器PC版安装需求

**最低系统要求：**

CPU：英特尔酷睿 i5-7400 或 AMD 等效

内存： 8 GB

GPU：NVIDIA GeForce GTX 960 或 AMD 等效

操作系统： Windows10 64 位

**推荐系统要求：**

CPU：英特尔酷睿 i5-9400F 或 AMD 等效

内存： 16 GB

GPU：NVIDIA GeForce GTX 1660 Ti  或 AMD 等效

操作系统： Windows10 64 位
